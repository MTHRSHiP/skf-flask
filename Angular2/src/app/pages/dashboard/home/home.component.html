Security Knowledge Framework

SKF is an expert system application that uses the OWASP (M)ASVS with detailed code examples (secure coding principles) to help developers getting the right security requirements and create applications that are secure by design.

Knowledge Base

All the information which is correlated throughout the project can be found here! Need specific information on the spot then search through the Knowledge base items.

Code examples

The code examples show how to approach some common functionalities and how to apply the secure coding principles! We have secure code examples in different languages for example; PHP, Python, Java and .NET

SKF Labs

We created in SKF Labs for Developers and security specialist to train their security testing skills. These labs re from basic to advanced and hard-core security issues. We also provided write-ups for the labs to teach the correct testing methodology.
